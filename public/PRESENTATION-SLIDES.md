## СЛАЙД 5: Проблема 1 — Согласованность интерфейса (4/5)

### Что не так:

- 🔴 **Gantt-шкала турниров** отличается от карточного стиля других разделов
- 🔴 **Разная типография и плотность** в Studies и Puzzles
- 🔴 **Таблицы лидеров** выбиваются из парадигмы

### Как это влияет на пользователя:

> Новичок может растеряться при переходе между разделами, т.к. визуальный язык меняется неожиданно.

### Решение:

→ **Добавить карточный режим турниров** (альтернатива Gantt)  
→ **Унифицировать типографику** (размеры, вес, line-height)  
→ **Синхронизировать стили таблиц** (лидеры, поиск, история ходов)

---

## СЛАЙД 6: Проблема 2 — Доступность (4/5)

### Что не так:

- 🔴 **Иконки без видимых подписей** — только aria-label, пользователь видит иконку
- 🔴 **Gantt-шкала с мелкими целями для тапа** — сложно нажать на мобиле
- 🔴 **Переключатель языка скрыт** — находится только в меню Preferences
- 🔴 **На мобильных элементы < 44px** — WCAG рекомендация не соблюдена

### Как это влияет на пользователя:

> Люди с нарушениями зрения, недостаточным знанием языка интерфейса или с маломобильностью испытывают сложности.

### Решение:

→ **Добавить видимые подписи и tooltips** к иконкам  
→ **Вынести переключатель языка в шапку** сайта  
→ **Увеличить целевые области** для мобиля до 48×48px  
→ **Провести аудит по WCAG 2.2 AA**

---

## СЛАЙД 7: Проблема 3 — Поддержка обучения (4/5)

### Что не так:

- 🔴 **Прогресс гостя не сохраняется** между сессиями → обнуляется при выходе
- 🔴 **Studies смешивает официальное и пользовательское** контент (даже развлекательный)
- 🔴 **Нет явного разделения по уровню** сложности (новичок / средний / продвинутый)
- 🔴 **Сложно вернуться к прерванному курсу** — нет виджета на главной

### Как это влияет на пользователя:

> Новичок теряет мотивацию, не может найти релевантный контент, забывает, где остановился.

### Решение:

→ **Сохранять прогресс в localStorage** для гостей  
→ **Разделить Studies** на "Официальные" и "Сообщество"  
→ **Добавить фильтр по уровню** с визуальными иконками  
→ **Создать виджет "Продолжить обучение"** на главной  
→ **Добавить warning** "Прогресс не сохранится без аккаунта"

## СЛАЙД 8: Концепт редизайна — Решение 1 (Согласованность)

### Карточный режим турниров

**БЫЛО (неправильно):**
```
┌─────────────────────────────┐
│ Gantt-шкала (сложная)       │ ← Выбивается
└─────────────────────────────┘
```

**СТАЛО (правильно):**
```
┌──────────────────────────────────────┐
│ [Карточный вид] [Gantt вид]          │
├──────────────────────────────────────┤
│ ┌──────────────┐ ┌──────────────┐   │
│ │ Арена 1-4    │ │ Арена 5-8    │   │
│ │ 15:30 Blitz  │ │ 16:00 Rapid  │   │
│ │ 24 участн.   │ │ 31 участн.   │   │
│ │ [→ Присо...]  │ │ [→ Присо...]  │   │
│ └──────────────┘ └──────────────┘   │
│ ┌──────────────┐ ┌──────────────┐   │
│ │ Классика     │ │ Турнир...    │   │
│ └──────────────┘ └──────────────┘   │
└──────────────────────────────────────┘
```

### Унификация типографики

| Элемент | БЫЛО | СТАЛО |
|---------|------|-------|
| Gap между карточками | 12px, 16px, 20px | Единый **16px** |
| Border-radius | разный | Единый **8px** |
| Размеры шрифтов | несистемно | 12px, 14px, 16px, 24px |

## СЛАЙД 9: Концепт редизайна — Решение 2 (Доступность)

### Видимые подписи к иконкам

**БЫЛО (неправильно):**
```
┌──────────────────┐
│ [◀] [▶] [⬛] [⚙] │  ← Только иконки!
└──────────────────┘
```

**СТАЛО (правильно):**
```
┌────────────────────────────────────┐
│ [◀ Пред.] [▶ След.] [⬛ Flip] [⚙] │
│ (Hover показывает tooltip)         │
└────────────────────────────────────┘
```

### Язык в шапке

**БЫЛО:**
```
┌──────────────────────────────────────┐
│ Lichess  ИГРА  ЗАДАЧИ  ... [⚙]       │
│                          ↓
│                     Settings→Language
└──────────────────────────────────────┘
```

**СТАЛО:**
```
┌──────────────────────────────────────┐
│ Lichess  ИГРА  ЗАДАЧИ  ... [🌐▼] [⚙]│
│                          ↓ EN/RU/FR
└──────────────────────────────────────┘
```

### Размеры целей для мобиля

- **Минимум:** 48px × 48px (WCAG рекомендация)
- **Расстояние между целями:** 8px минимум
- **На Gantt-шкале:** визуально расширить элементы
